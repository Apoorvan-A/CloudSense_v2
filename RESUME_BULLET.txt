CloudSense — Predictive Cloud Resource Analytics Engine
(Python, PyTorch, scikit-learn, Pandas, AWS CloudWatch, SageMaker)

  • Built a CEEMDAN + CNN-BiLSTM deep learning pipeline that decomposes
    non-stationary AWS EC2 workload signals into frequency components and
    trains per-IMF sub-models, achieving a 2-hour proactive scaling horizon
    on real Numenta NAB AWS CloudWatch data

  • Benchmarked 4 deep learning architectures (LSTM, CNN-LSTM, Bi-LSTM,
    Transformer) in PyTorch with early stopping, learning rate scheduling,
    and GPU acceleration; proposed model achieved best MAE and R² across
    all baselines

  • Outperformed 5 recent peer-reviewed methods (Bansal & Kumar 2023,
    Bi et al. 2023, Xing 2024, Lackinger et al. 2024, Vasumathi 2025)
    on CPU utilisation forecasting; CEEMDAN preprocessing reduced RMSE
    by up to 21% vs. standard LSTM

  • Designed full AWS 3-tier deployment architecture (Kinesis Data Firehose
    → S3 → Glue → SageMaker) for real-time inference via API Gateway
    integrated with Kubernetes HPA for automated pod scaling
